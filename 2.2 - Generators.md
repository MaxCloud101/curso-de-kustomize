# Generators

Los Generators crean recursos Kubernetes automáticamente a partir de archivos, variables o literales, evitando escribir YAML manualmente.

Los principales generators son:

- **ConfigMap Generator:** Genera un ConfigMap desde archivos o valores.
- **Secret Generator:** Genera Secrets automáticamente.
- **Helm Chart Generator:** Permite renderizar charts Helm desde Kustomize.
- **Exec / Plugin Generators:** Puedes crear generators personalizados.
