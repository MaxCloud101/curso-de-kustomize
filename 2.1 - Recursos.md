# Recursos

Resources es una sección del kustomization.yaml que indica qué manifiestos Kubernetes quieres incluir en el build final.

Piensa en:

- resources = manifiestos base
- transformers = modificaciones sobre esos manifiestos
- generators = crean manifiestos nuevos

## ¿Qué tipos de recursos puedo declarar?

- Cualquier recurso Kubernetes válido.

```
resources:
- deployment.yaml
- statefulset.yaml
- daemonset.yaml
- job.yaml
- cronjob.yaml
- replicaset.yaml
- pod.yaml
```

- CRDs (Custom Resources)

```
resources:
- prometheus.yaml
- argo-rollout.yaml
- certificaterequest.yaml
```

- Carpetas

```
resources:
- ../../base
```

- Urls Remotas

```
resources:
- https://github.com/MaxCloud101/curso-de-kustomize.git//examples/project1.1/base?ref=main
```
