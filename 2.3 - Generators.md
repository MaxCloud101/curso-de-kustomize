# Generators

Los Generators crean recursos Kubernetes automáticamente a partir de archivos, variables o literales, evitando escribir YAML manualmente.

Los principales generators son:

- **ConfigMap Generator:** Genera un ConfigMap desde archivos o valores. (ej. project2.2.1)
- **Secret Generator:** Genera Secrets automáticamente. (ej. project2.2.2)
- **Helm Chart Generator:** Permite renderizar charts Helm desde Kustomize. (ej. project2.2.3)
- **Exec / Plugin Generators:** Puedes crear generators personalizados.
