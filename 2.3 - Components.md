# Components

Los Components en Kustomize son módulos reutilizables que contienen configuraciones opcionales y pueden combinarse con distintos overlays. Sirven para evitar duplicar configuraciones comunes.

Piensa así:

- Base        = aplicación principal
- Overlay     = ambiente (dev/prod)
- Component   = funcionalidad opcional reutilizable

Ejemplos típicos de components:

- Logging
- Monitoring
- Sidecars
- Network Policies
- Labels corporativos
- Configuración de seguridad
- Service Mesh (Istio)
- Recursos comunes (requests/limits)
