# Transformers

Los Transformers son mecanismos que modifican recursos YAML automáticamente.

```
YAML original
    ↓
Transformers
    ↓
YAML transformado
```

Los transformers NO crean recursos nuevos.

Lo que hacen es:

- modificar
- agregar
- reemplazar
- transformar

partes del YAML.

## Transformers más usados

- namePrefix: Agrega prefijo (ej: project2.1)
- nameSuffix: Agrega sufijo (ej: project2.1)
- namespace: Cambia namespace (ej: project2.1)
- labels: Agrega labels (ej: project2.1)
- commonAnnotations: Agrega annotations (ej: project2.1)
- images: Cambia imágenes (ej: project2.1)
- replicas: Cambia replicas (ej: project2.1)
- replacements Transformer: copiar valores de un recurso hacia otro recurso (ej. project2.2)
- patches Transformer: modificar partes específicas de un recurso YAML (ej. project2.3)
- Transformer Custom: permite crear transformaciones personalizadas sobre YAML. (ej. project2.4)
